1: What is the differences between var, let and const?

Before ES6, var was the most used keyword to declare variables. But day by day, developers were facing many problems with 'var' as it hoists declaration at the top. Even it has no block-scope which is most annoying to many developers.

After the innovation of ES6, var has no longer necessity as let and const have better effect.

Let and Const are almost same withoug one point: the variable declared with let can be reassigned but in case of const, it's impossible.

2: What is the differences between Arrow function and Regular function?

There are a lot of ways to declare functins in javascript. One of them is regular function and the other one is arrow function.

A regular function can easily construb objects while an arrow function can't be used as a constructor

We know, arrow function of just one line expression is called inline arrow function. It returns the value implicitly. It's a special case which doesn't present in regular function. Because, if you omit return statement in regular function, it returns implicitly 'undefined'!
